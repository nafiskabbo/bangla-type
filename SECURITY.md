# Security policy

## Supported versions

We release security updates for the current major/minor line. Check [Releases](https://github.com/nafiskabbo/bangla-type/releases) for the latest version.

## Reporting a vulnerability

If you believe you’ve found a security vulnerability, please report it responsibly:

1. **Do not** open a public issue for security-sensitive bugs.
2. Email the maintainers or open a **private** security advisory:  
   [GitHub → Security → Advisories → New draft advisory](https://github.com/nafiskabbo/bangla-type/security/advisories/new).
3. Include a clear description, steps to reproduce, and impact if possible.
4. We will acknowledge and respond as soon as we can. We may ask for more details and will keep you updated on a fix and release.

We appreciate your help in keeping BanglaType and its users safe.
