# Keyboard layouts

BanglaType ships only **Bangladeshi** layouts. Indian layouts (Inscript, Disha, etc.) are out of scope.

All use **Avro Keyboard Layout v5** (`.avrolayout` JSON).

| # | Layout            | বাংলা নাম     | Type     | Author / Origin              |
|---|-------------------|---------------|----------|------------------------------|
| 1 | Avro Phonetic     | অভ্র ফোনেটিক  | Phonetic | OmicronLab, Dhaka 🇧🇩        |
| 2 | Probhat           | প্রভাত        | Fixed    | Bangladeshi Linux community  |
| 3 | Munir Optima      | মুনীর অপ্টিমা | Fixed    | Munir Chowdhury 🇧🇩          |
| 4 | Avro Easy         | অভ্র ইজি      | Fixed    | OmicronLab, Dhaka 🇧🇩        |
| 5 | Bornona           | বর্ণনা        | Fixed    | The Safeworks 🇧🇩            |
| 6 | National (Jatiya) | জাতীয়        | Fixed    | BCC 🇧🇩                      |
| 7 | Akkhor            | অক্ষর         | Fixed    | Khan Md. Anwarus Salam 🇧🇩   |

**Custom layouts:** Put `.avrolayout` files in `~/Library/Application Support/BanglaType/Layouts/` — they appear in the layout switcher.
