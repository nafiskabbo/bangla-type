# Keyboard shortcuts

| Shortcut              | Action                         |
|-----------------------|--------------------------------|
| ⌃ Control + Space     | Toggle Bangla ↔ English        |
| ⌃ Control + \         | Layout switcher HUD             |
| ⌃ Control + 1 … 7     | Switch to layout 1–7           |
| ⌥ Option + Backspace  | Delete যুক্তাক্ষর cluster       |
| ⌘ Cmd + Backspace     | Delete composing word           |
| Esc                   | Cancel composing                |
| ⌃ Control + ,         | Preferences                     |
| ⌃ Control + K         | On-screen keyboard              |

All shortcuts can be changed in **Preferences → Shortcuts**.
